# vidly api

This is a simple video rental app that I built while learning NodeJS. It allows users to rent videos and manage their account.

## Installation

1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Features

- Register and login
- View all movies
- Rent movies
- View rented movies
- Return rented movies
- View account details

## Technologies

- NodeJS
- Express
- MongoDB
- Mongoose
- Joi
- JWT
- Bcrypt
